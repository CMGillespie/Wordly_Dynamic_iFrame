Overview

This implementation enforces a "Single Point of Control" by using the show parameter to restrict the internal Wordly menu.

Surgical URL Parameters

  • lang: Directs the translation engine to the target locale.
  • show: Limits internal menu options to only the selected language.
  • finalOnly: (APAC Specific) Optimizes display for character-based sentence structures.
