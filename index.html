Technical Implementation Guide

Architectural Overview

This implementation utilizes a Parental Control Model for iframe integration. By offloading language selection and UI state management to the host application, developers can ensure a consistent UX while leveraging Wordly's translation engine. This approach is favored by NetSec teams as it minimizes user interaction with third-party iframe elements and enforces strict session parameters.

Surgical Parameter Logic

We leverage specific query string parameters to suppress native Angular Material components inside the attend.wordly.ai domain:

ParameterTypeSecurity/UX Impact
langString (ISO)Initializes the translation engine into a specific locale immediately.
showString (ISO)Critical: Limits the internal Mat-Select menu to a single option, effectively disabling the selector bar.
speakerLanguageBooleanSuppresses the "Speaker is speaking..." toast notification to reduce UI clutter.
finalOnlyBooleanOptimizes character-based rendering for APAC regions by preventing partial sentence updates.
